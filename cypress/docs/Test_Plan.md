# Plan de Pruebas

## 1. Introducción
Este documento describe el plan de pruebas para validar el flujo de búsqueda y selección de vuelos en el sitio web de Wingo.

## 2. Objetivo
Verificar que los usuarios puedan buscar y seleccionar vuelos correctamente en la ruta Bogotá – Cali con diferentes configuraciones de fechas y pasajeros.

## 3. Alcance
Las pruebas cubrirán:

- Búsqueda de vuelos
- Selección de vuelos de ida y regreso
- Selección de tarifa económica (Go Basic)
- Validación de fechas entre meses diferentes
- Selección de pasajeros

## 4. Tipos de pruebas

| Tipo de prueba | Descripción |
|---|---|
| Funcionales | Validar que el flujo funcione correctamente |
| Exploratorias | Identificar comportamientos inesperados |
| UI | Validar elementos visuales |

## 5. Entorno de pruebas

| Elemento | Configuración |
|---|---|
| Navegador | Chrome |
| Sistema Operativo | Windows 10 |
| Sitio web | https://www.wingo.com |

## 6. Criterios de entrada

- Sitio disponible
- Datos de prueba definidos
- Casos de prueba documentados

## 7. Criterios de salida

- Casos de prueba ejecutados
- Evidencias registradas
- Bugs reportados
